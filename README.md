# springCloudDemoProject
#####SpringCloudDemo&lt;&lt;create on IntelliJ IDEA>>
    Some demofunctions of SpringCloud.
##api_gateway 
    Zuul网关，实现URL转发
##eureka
    Eureka注册中心
##feign_consumer 
    feign消费者，实现服务提供者生命调用
##hello_service_api
    hello_service实体类无需运行，只需在运行前构建
##hystrix 
    结合@HystrixCommand实现方法监控窗口
##ribbon 
    实现ribbon服务提供者
##ribbon_consumer
    实现ribbon服务消费者
##turbine
    实现集群监控
